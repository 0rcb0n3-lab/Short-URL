# Short URL
Cкрипт для сокращения ссылок через сервис `clc.li` и просмотра количества кликов по уже сокращённым ссылкам.

## Функционал
- **Сокращение длинных ссылок** → получение короткой версии (`https://clc.li/Qwerty`)
- **Просмотр статистики кликов** по существующей короткой ссылке
- Автоматическое определение типа ссылки (длинная или короткая)

## Требования
- Python 3.6+

  - `requests`
  - `os`
  - `python-dotenv`
  - `urllib.parse`

### Как использовать
Введите ссылку:
- Если ввести обычную ссылку (например, `https://yandex.ru`), она будет сокращена.
- Если ввести короткую ссылку (например, `https://clc.li/qwerty`), будет показано количество кликов.

#### Примеры:

- Введите ссылку: `https://example.com/very/long/path` ->
Короткая ссылка: `https://clc.li/xyz789`
- Введите ссылку: `https://clc.li/xyz789` ->
URL: `https://clc.li/xyz789` total clicks: 42
